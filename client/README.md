# 409_USW_Campus Client (using React Native)

### 테스트
1.
      client 폴더로 이동
      
      ```
      cd client
      ```
2.
      build
      
      ```
      npm start
      ```
 3.
      시뮬레이터 실행
      
       1. Expo Go 애플리케이션 설치 (앱스토어 혹은 플레이스토어 사용)
       2. 카메라 앱을 이용해 콘솔창에 나타난 QR코드 촬영
       3. 저장(단축키 Ctrl + S or Command + S) 할 때마다 실시간으로 반영 확인
